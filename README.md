# 📊 What’s in a Name? Workforce Taxonomy Analysis

What’s in a Name? is a labor market intelligence and workforce taxonomy analysis project designed to explore how organizations classify, title, and segment analyst and data-oriented roles across the labor market. Built using scraped LinkedIn job postings, the framework combines automated data collection, title parsing, and dashboard visualization to examine inconsistencies in workforce taxonomy, job architecture, and labor market signaling.

The project supports downstream workforce strategy analysis by transforming fragmented job posting data into a structured dataset suitable for labor market interpretation, dashboarding, and role classification analysis.

## 🧩 File Overview

| File Name                           | Purpose                                                              |
| ----------------------------------- | -------------------------------------------------------------------- |
| LinkedIn Detail Scraping.ipynb      | Automated LinkedIn scraping and structured extraction logic          |
| LinkedIn Pull of Analytics Jobs.csv | Structured labor market dataset generated from scraped postings      |
| What’s in a Name Dashboard.pbix     | Interactive Power BI dashboard exploring workforce taxonomy patterns |
| What’s in a Name Dashboard.pdf      | Exported dashboard view for executive-style review and sharing       |

## 🔍 Key Features

### Workforce Taxonomy Analysis

Explores inconsistencies and overlap across analyst, engineering, data science, and product-oriented job classifications.

### Labor Market Segmentation

Identifies patterns in employer hiring behavior, role clustering, and title distribution across organizations.

### Automated Data Collection

Uses Python-based scraping and parsing workflows to generate structured labor market datasets.

### Dashboard Visualization

Transforms extracted labor market data into interactive workforce intelligence and visualization artifacts.

### Classification Exploration

Highlights challenges in title standardization, workforce taxonomy alignment, and semantic role ambiguity.

## 🛠 Methodology

Collect job posting data through automated scraping workflows.

Parse and structure labor market data into standardized analytical datasets.

Visualize workforce classification patterns using Power BI dashboards.

Analyze title clustering, employer relationships, and labor market segmentation trends.

## 💡 Ideal Use Cases

Workforce planning and labor market analysis

Job architecture and title standardization discussions

Compensation benchmarking and labor segmentation

Workforce taxonomy and organizational design analysis

Analytics and workforce intelligence portfolio demonstration

## 📌 Notes

Dataset reflects publicly available job posting information collected for analytical and educational purposes.

Project focuses on workforce taxonomy interpretation and labor market structure rather than predictive modeling.

Framework can be expanded to include skills clustering, NLP-based title normalization, or broader labor market segmentation analysis.
